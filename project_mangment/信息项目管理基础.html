# PMO是什么,有什么关键特征

pmo是信息项目管理办公室,主要职责为:

  1. 在pmo管理的所有项目之间进行共享和协调资源
  2. 明确和制定项目的管理方法、最佳实践和标准
  3. 负责制定项目方针、流程、模版和其他共享资料
  4. 为所有项目进行集中的配置管理

# 单个项目管理过程

# PDCA 和 五大过程组的关系

5大过程组就是根据 PDCA 的理念 来划分项目管理的过程(47个过程,10大领域)

# 项目管理的过程组

  1. 启动过程组 只有两个过程:
    • 制定项目章程 (整体管理领域)
    • 识别干系人 (干系人管理领域)
  2. 计划过程组
  3. 执行过程组
  4. 监控(监督与控制)过程组 (对应PDCA中的 CA)
    • 监控监督和控制
  5. 收尾过程组
    • 结束项目阶段 (整体管理领域)
    • 结束采购 (采购管理领域)
